# AIM Visual Toolkit

Swiss Design visual metaphors for AI education.

## Features

- **24+ Visual Metaphors** — Hand-crafted SVG illustrations for AI concepts
- **AI Generator** — Create custom metaphors using Gemini AI
- **Export Options** — Copy SVG, download SVG/PNG (4x resolution)
- **Swiss Design** — Clean, minimal, geometric aesthetic

## Tech Stack

- React 19 + TypeScript
- Vite 7 + Tailwind CSS 4
- Framer Motion
- Netlify Functions (OpenRouter API)

## Development

```bash
npm install
npm run dev
```

## Environment

Create `.env` file:

```
OPENROUTER_API_KEY=sk-or-v1-your-key-here
```

## Build

```bash
npm run build
```

## Deploy

Automatic deployment via Netlify on push to `main`.
